    # java的变量

    • java中也可以自定义变量，每一个java变量属于一种类型，也就是说必须指明java变量的类型
    • 声明示例：
    double salary；
int count;
    1
    2
    • 变量的声明属于一条完整的语句，用分号结束。
    • 变量名的命名规则：

    必须以字母或者下划线开头，由字母或者数字构成的序列。

    java的变量不能使用java的保留字进行命名。

    java的变量名是区分大小写的，如：Count 和count两个变量是不相同的。

    长度没有限制，注意java的变量命名规范应当是变量具有可读性，这是一个合格的java开发的基本功。

    # 变量初始化

    • java的变量在使用前必须初始化。
    • 可以在定义时初始化，也就是在定义的时候使用赋值语句进行初始化，这里先说一些，java的赋值符号和大多语言一样是“=”
    • 例如：int count=12；
    • 或者在定义完毕之后在使用前进行初始化也是可以的。
    • 例如：
    int count;
count=12;//在使用之前必须初始化，否则就会报错
    1
    2
    • java 的变量声明可以放在代码的任何地方，放在不同的地方的作用域是不一样的，我们后面会说到。

    • 类常量：在final前面加上一个static关键字，并将定义语句写在类中main方法之外。便可以将常量的作用域提升至整个类。

    # 运算符

    • 算数运算符

      • " +" 、"-"、 "*"、 "/"表示加减乘除运算符
      • 还有一个特殊的“%”叫做取余运算符
      • 当“/”运算符的两个操作数都是整数的时候，表示整数除法，否则表示浮点除法
      • 整数除以0会产生一个异常，浮点数除以0会得到无穷大后者NaN结果

    • 自增运算符和自减运算符

      • 自增++，自减--
      • 如果是n++，表示先使用n的原值，再进行加一操作，++n表示先加一再取值
      • 自减运算符合自增运算符一样

    • 关系运算符与boolean运算符

      • 关系运算符：==（相等）、<（小于） 、>（大于） 、>=、<=、!=（不等于）、&&（与）、||(或)、!(非)
      • 注意：在使用&& 、||关系运算符的时候，比如：表达式1 && 表达式2;例如：(2>3) && (3<2);这时候，运算第一个表达式的结果为false，则不会再去计算第二个表达式，直接返回false了，因为 && （与）运算必须两个表达式或者变量同时为true的时候，结果才会为true。
      • 三元运算符：condition ? expression1:expression2；当condition为真时计算第一个表达式作为返回结果，否则计算第二个表达式作为返回结果。

    • 位运算符：

      • 按位或：&
      • 按位与：|
      • 异或：^
      • 非：~
      • >> 二进制位右移
      • << 二进制位左移
      • >>>运算符将用0填充高位，>> 运算符用符号位填充高位。没有<<<运算符。

    # 数值类型之间的转换

    • 在基本数据类型之间的转化中，部分的转换是由精度损失的，比如一个浮点数转换成为一个整数则会损失精度。
    • 自动转换：在java中会有一个类型自动转换存在，例如当另个操作数运算的时候，一个整型一个浮点型，则编译器会自动将整型转换成为浮点型进行计算，结果也是浮点型
    • java中的自动转换顺序是：范围小的会自动向范围大的转，否则就需要强制转换
            int x=1;
        double v = x + 1.2;
        
        
        int x=1;
        int v = x + 1.2;
        就会报错

    • 强制转换
      • 当范围大的数据向范围小的数据转换时就需要用到强制类型转换，否则就会报错，也就是必须显式的强制转换
      • 强制转换的格式是【 转换后的数据类型 变量名 = (强制转换的类型)(需要被转换的表达式/数据变量)
            int x=1;
        int v = (int) (x + 1.2);

    # 枚举类型

    • 变量的取值只在几个有限的集合中，例如：销售的服装只有大中小超大这四个尺寸
    • 所以java针对这种数据就使用枚举来定义
    • 枚举的定义方式，可以在成员的位置进行枚举的定义
        enum Size{SMALL,MEDIUM,LARGE};
    Size size=Size.MEDIUM;

    SMALL,LAGER,MEDIUM;

    public static void main(String[] args) {
        System.out.println(Size.SMALL);
    }
