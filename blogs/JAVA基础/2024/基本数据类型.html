    基本数据类型

    蓝同学 9/9/2024 JAVA

    # java数据类型

    • java是一个强类型语言，必须为每一个变量声明一种类型，java一共有8种基本类型，4种整型、2种浮点类型、1种用于表示Unicode编码的字符单元的字符类型char,还有一种Boolean类型的
    • java还有一个能够计算任意精度的算术包，通常称为大数值，他不是java的基本数据类型，而是一个对象。

    # 整型

    • 就是数学上的整数类型，正整数、负整数
    类型 大小
    int 4字节
    short 2字节
    long 8个字节
    byte 1字节
    • java和C++的不同在于，每一种数据类型的大小都是固定的，在不同的平台上的大小都是固定的。
    • 长整形数据long后面有一个后缀L，如：4000000000L
    • 十六进制数据的前缀：0x
    • 八进制数据前缀0
    • 二进制数据的前缀：0b
    • 在java7开始可以用下划线分隔数字，java会自动去除，如：1_000_000

    # 浮点类型

    类型 大小
    float 4字节
    double 8字节
    • float数据类型有一个后缀F,如3.14F
    • 没有后缀的浮点数默认是double类型的，也可以给浮点数加上一个后缀D显式的指明是一个double类型数据。
    • 浮点数的运算规则遵循IEEE 754，浮点数运算的结果可能会出现以下三种
    1. 正无穷大
    2. 负无穷大
    3. NaN（不是一个数字）
    • 这三种类型在Double类中是三个常量值：
    public static final double POSITIVE_INFINITY = 1.0 / 0.0;

public static final double NEGATIVE_INFINITY = -1.0 / 0.0;

public static final double NaN = 0.0d / 0.0;

    但是不能够通过检测判断一个数是不是这三个常量 任何可能出现这三种结果的计算都会抛出异常

    # char型

    • 表示单个字符，如'A',但是"A"则表示一个字符串。

    Uncoide编码：

    • Unicode编码单元可以表示成十六进制数据,他的返回时从\u0000~\Uffff,如：\u03C0表示字母pi，可以直接使用Unicode的十六进制来表示单个字符。
    • 除了使用Unicode字符编码外，还可以在字符常量或者字符串中使用转义字符，在java中有一些转移字符也可以表示一些特殊的字符
    转义符 名称
    \b 退格
    \t 制表
    \n 换行
    \r 回车
    \" 双引号
    \' 单引号
    \\ 反斜杠
    • 想要学习char类型的数据，就必须了解Unicode编码
    • 在没有Unicode编码之前，不同的国家的字符编码机制都不相同，对于任意给定的字符或者代码值，不同的编码方案有不同的编码和表示不同的字符。
    • 还有就是不同的编码方案，对字符采用的编码长度也不一致
    • 所以Unicode就是来解决这些问题的
    • java中，char类型用UTF-16编码表示一个代码单元。

    # boolean类型

    • boolean类型有两个值：false和true，用来判断逻辑条件，整形和布尔之间不能进行相互转换。

    # 各种数据类型的定义格式如下：

            int x=10;
        short y=1;
        long z=400000L;
        byte a=1;

        char b='a';
        String c="abc";
        float d=3.14F;
        double e=3.24441D;
        boolean f=true;
        
        输出：
        System.out.println(Arrays.asList(x,y,z,a,b,c,d,e,f));

控制台输出：
[10, 1, 400000, 1, a, abc, 3.14, 3.24441, true]
