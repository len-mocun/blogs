    # java 控制流程

    • 和任何程序设计语言一样，java使用条件语句和循环结构确定控制流程

    # 块作用域

    • 块（复合语句）：是指由一对花括号括起来的若干条简单的java语句，块确定了变量的作用域，一个块可以嵌套在另一块中
    public static void main(String[] args){
    ……main方法的块
    {
        ……另一个块
    }
}

    # 循环

    while(条件){ …… };当条件成立就会一直重复执行块中的代码，当条件不成立则不执行

do{
    ……
} while(条件);//这一个会先执行一次再进行条件的判断，如果条件成立则会继续重复执行do中的代码，这种循环特点就是无论条件成不成立都会执行一次do中的代码。

    # 确定循环，就是已经知道了循环的次数，使用for循环

    for(计数器的初始化 ; 每次循环需要检验的条件 ; 如何更新计数器){
    
}

例如：
for(int i = 0 ; i <= 10 ; i++){
    ……
}
表示计数器从i==0开始，每次循环都会判断i是否小于等于10，如果满足就会执行代码块，每次i都会步进1。

    # 多重选择switch语句

    • 多重选择可以使用if()else{};来实现但是，这种实现条件比较多的时候，逻辑复杂，代码风格也不是很好，而且不容易阅读和维护
    • java同样也有一个支持多重选择的选择分支语句switch
    switch(需要判断的数据){
    case 值1:
    ……
    break;
    case 值2:
    ……
    break;
    case 值3:
    ……
    break;
    default:
    ……
}

    • 表示判断switch中的数据可以是变量，变量的类型可以是char、byte、short、int以及他们的包装类，jdk7之后可以是字符串变量了。
    • 其中值是相应的变量的可能的值，如果匹配就会跳到相应的case代码块执行相应的操作，最后break表示跳出switch语句，如果没有break则会进入下一个条件中执行，直到遇到break。
    • 最后的default表示如果值都不匹配会走这一个默认的操作块。

    # 中断控制

    • 在循环中可以使用中断控制来控制程序的循环流程
    while(years <= 100){
    ……
    if(某个需要跳出循环的条件) break;//当程序出现跳出循环的条件就会跳出循环去执行循环下面的代码
}
    1
    2
    3
    4
    • 另外，还有一个叫做continue，它表示终止当前的这次循环，进入下一次的循环。
    • 在java中，设计者将goto 作为了保留字，但实际中并没有打算使用它，因为goto语句被称为是一种拙劣的程序设计风格。但是偶尔使用goto进行跳转还是很有益处的。java设计者也同意这种看法，所以java中增加了一条带标签的break语句，来支持这种编程风格
    
again(这个是标签名，可以自定义):
while(years <= 100){
    ……
    if(某个需要跳出循环的条件) break again;//当程序出现跳出循环的条件就会跳出循环
    并跳转到指定的标签处。
}
